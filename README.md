# LastEpochBuildSynchronizeTools

这是一个用于将《最后纪元》的采蘑菇配装器上的BD一键同步到游戏中的离线存档的工具。

# 用法

1. 安装油猴脚本
2. 打开配装器进行配装，记得点击“保存/分享”刷新当前的链接
3. 点击“导出BD”按钮复制BD数据
4. 运行entry.py（需PyQt5库支持），刷新存档并选择一个存档进行修改
5. 将BD数据复制到下方并点击修改存档
6. 打开游戏游玩
